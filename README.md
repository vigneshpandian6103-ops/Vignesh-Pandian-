# DecodeLabs Responsive Frontend Interface (Project 1)

This project forms the foundational architecture module for the DecodeLabs Industrial Training Program (Batch 2026).

## 🛠️ Tech Stack & Constraints
- **Languages:** HTML5 (Semantic Structure), CSS3 (Mobile-First Grid/Flexbox), Vanilla JS.
- **Framework Constraint:** Developed with 100% Core Web Basics (Strictly no React, Vue, or TailwindCSS).
- **Compliance Rules:** Follows 2025/2026 Warm Grounded Aesthetics and WCAG accessibility integrity.

## 📈 Execution Pipeline Followed
1. **Discovery:** Established layout flow architectures.
2. **Wireframe:** Built for viewport hierarchies in grayscale scaling.
3. **Semantics:** Landmark elements map mapping (`<header>`, `<main>`, `<article>`).
4. **Style Layouts:** Adaptive responsive scaling using fluid typography rules (`clamp()`).
